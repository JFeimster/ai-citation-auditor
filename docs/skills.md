# AI Citation Auditor Skills

## Core Skills

### Citation Readiness Audit
Reviews whether finance content is structured clearly enough for AI answer engines to understand, summarize, and extract.

### Answer Block Detection
Identifies missing direct-answer sections, definition blocks, comparison blocks, and summary sections.

### Risky Claim Rewrite
Flags overconfident finance, lending, funding, credit, or outcome-based claims and suggests safer rewrites.

### H2 and Page Structure Review
Improves headings so the page is easier for readers, search engines, and AI systems to parse.

### Table Recommendation
Suggests tables for comparisons, eligibility criteria, fees, timelines, definitions, pros/cons, and decision factors.

### FAQ Rewrite
Turns vague FAQs into specific, answer-ready finance questions and responses.

### Schema Fit Review
Recommends possible schema types based on page intent, such as FAQPage, Article, BreadcrumbList, Organization, or SoftwareApplication.

### Internal Link Strategy
Suggests links to related explainers, comparison pages, service pages, glossary pages, and conversion pages.

### CTA Alignment Review
Checks whether the CTA matches user intent, page stage, and financial-decision sensitivity.

### Publish Checklist
Provides a final pre-publish checklist before the content goes live.
