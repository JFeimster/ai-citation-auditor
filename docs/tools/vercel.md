# Vercel Deployment Notes

## Static deployment
This project can deploy as a static site with no build command.

## Suggested settings
- Framework preset: Other
- Build command: none
- Output directory: root

## Future upgrade path
Move to Next.js only if the project needs:
- Dynamic audit intake forms
- User accounts
- Saved audit history
- API routes
- AI SDK integration
- Server-side lead capture
- Programmatic SEO pages

## Do not upgrade just for styling
The current static version is enough for a launch page, lead magnet, and small SEO hub.
