# Jules Instructions

## Best Jules tasks
- Fix broken links
- Add documentation
- Create new static pages
- Update metadata
- Improve mobile responsiveness
- Create a pull request for review

## Repo expectations
- Keep changes small and reviewable.
- Do not alter brand positioning without instruction.
- Do not add dependencies.
- Preserve existing CTA URL.
- Run a basic browser check after changing layout files.

## Preferred branch names
- docs/update-readme
- page/add-services
- page/add-checklist
- fix/mobile-nav
- seo/add-schema
