# Tools

## ChatGPT
Primary access point for AI Citation Auditor.

## OpenAI Codex
Recommended for editing the static site, adding pages, refactoring HTML/CSS/JS, and generating pull requests.

## Google Jules
Useful for GitHub-based maintenance tasks, docs updates, bug fixes, and feature branches.

## Vercel
Recommended deployment option for static hosting and future Next.js expansion.

## Netlify
Simple drag-and-drop static deployment option.

## GitHub Pages
Free static hosting option for lightweight microsite deployment.

## Notion
Useful for storing audit frameworks, swipe files, risky-claim examples, and content checklists.

## Webflow / Framer / Wix / Carrd
Useful for embedding the landing page or CTA widget after deployment.
