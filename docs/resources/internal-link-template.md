# Internal Link Template

## Hub / Spoke Model
- Hub page: the broad topic page that collects the main theme.
- Spoke pages: narrower pages that answer one question, compare one idea, or support one decision.
- Cross-links: links between closely related spokes when they help the reader.

## Link Placement Checklist
- Add one primary CTA link near the top.
- Add one or two related links in the body.
- Add a related links block near the end.
- Link from broad pages to focused support pages.
- Link from comparison pages back to the main GPT CTA.
- Avoid stuffing the same link in every section.

## Cluster QA Checklist
- Each spoke links back to a hub.
- Each hub links out to the most useful spokes.
- Anchors describe the destination clearly.
- No orphaned pages remain in the cluster.
- Links support the reader path, not just SEO volume.
