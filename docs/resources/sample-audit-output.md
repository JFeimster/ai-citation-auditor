# Sample Audit Output

## Example Scorecard
- Citation readiness: 74 / 100
- Answer clarity: 16 / 20
- Structure: 14 / 20
- Trust signals: 12 / 20
- Risky claims: 10 / 15
- Schema fit: 11 / 15
- Internal links: 11 / 10

## Risky Claim Findings
- "Get approved fast" reads as a guarantee and should be rewritten.
- "Best option for every business" is too broad and should be fit-based.
- "Guaranteed results" should be removed.

## H2 Recommendations
- What is this page for?
- Who is this for?
- What criteria matter most?
- What should readers review before publishing?
- How should this page be structured for AI extraction?

## FAQ Recommendations
- What does citation readiness mean?
- Does this guarantee citations?
- Is this a compliance review?
- What should I fix first?

## Schema Recommendations
- FAQPage if the FAQ is visible.
- BreadcrumbList if breadcrumbs are visible.
- Organization only if the page is describing the brand.
- SoftwareApplication only for the GPT page.

## Internal Link Recommendations
- Link to the main GPT CTA.
- Link to the checklist page.
- Link to the risky claims rewrite guide.
- Link to the schema recommendation guide.

## Final Publish Checklist
- Clear answer block added.
- Risky claims removed or softened.
- FAQ visible and accurate.
- Schema matches the page.
- Internal links added.
- No guaranteed outcome language remains.
