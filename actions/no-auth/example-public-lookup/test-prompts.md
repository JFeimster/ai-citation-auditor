# Test Prompts

1. Look up a simple public query.
2. Search for a query that should return a clear single result.
3. Search for a query with an ambiguous term.
4. Search for a query that returns no result.
5. Search with an empty query.
6. Search for a query that includes extra punctuation.
7. Search for a query that tests a boundary around public-only content.
8. Search for a query that should surface the latest public update timestamp.
