# Fonts

This directory is reserved for licensed font assets if the project ever needs them.

Current status:
- The site uses system fonts and the existing CSS font stack.
- No proprietary font files should be committed here unless the license is confirmed.

Checklist before adding a font:
- Confirm the license allows repository distribution.
- Document the license source and usage restrictions.
- Keep file names descriptive.
- Update the CSS explicitly so the font swap is intentional.
