# UI Icons

This directory stores small reusable UI icons for the static site.

Use this folder for:
- Inline SVGs that are reused across pages.
- Optimized SVG assets for buttons, badges, and status markers.
- Small helper icons that should stay lightweight.

Guidelines:
- Prefer inline SVG for critical UI where possible.
- Keep icons original and licensed for repo use.
- Add a short note if an icon is adapted from an external source.
- Do not drop in third-party icon packs without license notes.
