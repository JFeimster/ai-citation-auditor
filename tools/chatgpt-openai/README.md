## How the pieces fit
- Prompt packs define the copy-paste prompts that operators and agents can reuse.
- Action planning docs explain how GPT Actions should be designed, tested, and bounded.
- Skill playbooks describe the repeatable processes behind the prompts.
- Integration playbooks show how outside systems map into registries and workflow docs.
- Registries store the structured data that these docs reference and validate.
