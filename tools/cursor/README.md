# Cursor

Purpose:
- Local static-site editing and fast iteration.

Use when:
- You are editing HTML, CSS, JS, or docs in the repo.

Do not use when:
- The task needs a deployment action or external API.

Safe usage rules:
- Stay within the repo map and do not overwrite unrelated edits.
- Prefer scoped file edits over broad rewrites.
- Use shared assets and page-specific files intentionally.
