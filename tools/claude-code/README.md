# claude-code

Purpose:
- Command-line implementation help and code review.

Use when:
- you need command-driven repo changes or review notes.

Do not use when:
- the work is already done and only needs a summary.

Safe usage rules:
- Prefer non-destructive commands.
- Keep the diff aligned with the request.
- Call out risks and missing checks.
