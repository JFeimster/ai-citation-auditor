# AI Citation Auditor Landing Page

A launch-ready static landing page for the **AI Citation Auditor** Custom GPT.

## Files

- `index.html` — landing page markup and copy
- `styles.css` — premium dark SaaS styling
- `script.js` — mobile nav, sticky header state, and copy-prompt button

## Preview locally

Open `index.html` in your browser.

## Customize

In `index.html`, replace:

- GPT URL if it changes
- Open Graph image URL
- Any service upsell links, booking links, or analytics comments
- Footer copy if you want to add a company or creator name

## Deploy

### Netlify
Drag the folder into Netlify Drop, or connect a GitHub repo and deploy as a static site.

### Vercel
Create a new project, import the folder/repo, and deploy with no build command.

### GitHub Pages
Upload these files to a repository, then enable Pages from the repository settings.

### Static hosting
Upload all files to the same public directory on your host.

## Embed in another site

### iframe
Use this after deploying:

```html
<iframe
  src="https://your-domain.com"
  title="AI Citation Auditor"
  width="100%"
  height="900"
  style="border:0; border-radius:24px; overflow:hidden;"
  loading="lazy">
</iframe>
```

### Wix / Framer / Webflow / Carrd
Add an Embed or Custom Code block and paste the iframe snippet above.

## Notes

This page intentionally avoids claims that the GPT guarantees rankings, AI citations, approvals, credit improvement, funding, or financial outcomes.
